# PathTracing

A Project on Computer Graphics

## How to Run

```
mkdir build
cd build
cmake ..
./rayTracing n_samples
```

## Debug Build

```shell
mkdir debug
cd debug
cmake -DCMAKE_BUILD_TYPE=Debug ..
```
